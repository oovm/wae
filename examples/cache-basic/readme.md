# WAE Cache 基础示例

本示例展示了 WAE 框架中缓存服务的基本使用方法。

## 功能说明

本示例演示了基本的缓存操作：

- **设置缓存 (SET)**: 存储键值对
- **获取缓存 (GET)**: 读取键值对
- **删除缓存 (DELETE)**: 移除键值对
- **检查存在 (EXISTS)**: 验证键是否存在
- **TTL 管理**: 设置和查询过期时间
- **清空缓存 (CLEAR)**: 移除所有键值对

## 运行方式

```bash
cargo run --package cache-basic
```

## 示例内容

1. **初始化缓存服务**
   - 创建缓存实例，设置默认 TTL

2. **设置缓存项 (无 TTL)**
   - 存储多个用户数据

3. **获取缓存项**
   - 读取已存储的缓存数据

4. **检查缓存是否存在**
   - 验证键的存在性

5. **设置缓存项 (带 TTL)**
   - 演示过期时间设置
   - 查询剩余过期时间

6. **更新缓存项**
   - 修改已有缓存值

7. **删除缓存项**
   - 移除指定缓存

8. **获取不存在的缓存**
   - 演示缓存未命中场景

9. **清空所有缓存**
   - 移除所有缓存项

## 注意事项

本示例使用内存数据结构模拟缓存服务，实际使用时需要：

1. 配置 Redis 或其他缓存后端
2. 设置连接参数和认证信息
3. 配置序列化格式
4. 设置合理的 TTL 策略

## 相关模块

- `wae-cache`: 缓存服务核心模块
