# WAE Monitoring Basic Example

This example demonstrates basic usage of the WAE Monitoring service.

## Features
- Basic monitoring setup
- Simple alert configuration
- Health check examples

## Requirements
- Rust 1.75+
- Tokio runtime

## Usage
1. Clone the repository
2. Run the example:
   ```bash
   cargo run
   ```

## Example Code
The example shows how to:
- Initialize the monitoring service
- Create basic alerts
- Check system health
- Monitor resource usage

## Configuration
The example uses default configuration values. You can modify the code to use your specific monitoring setup.

## Dependencies
- wae-monitoring - WAE Monitoring service
- tokio - Async runtime
- chrono - Date and time handling
