# WAE Database 基础示例

本示例展示了 WAE 框架中数据库操作的基本使用方法。

## 功能说明

本示例演示了基本的数据库 CRUD 操作：

- **CREATE (INSERT)**: 创建新记录
- **READ (SELECT)**: 查询记录
- **UPDATE**: 更新记录
- **DELETE**: 删除记录

## 运行方式

```bash
cargo run --package database-basic
```

## 示例内容

1. **初始化模拟数据库**
   - 创建用户和产品的示例数据

2. **查询所有用户**
   - 展示如何获取所有记录

3. **查询单个用户**
   - 展示如何通过 ID 查询记录

4. **创建新用户**
   - 展示如何插入新记录

5. **更新用户**
   - 展示如何更新现有记录

6. **查询所有产品**
   - 展示如何查询另一个数据表

7. **删除用户**
   - 展示如何删除记录

## 注意事项

本示例使用内存数据结构模拟数据库，实际使用时需要：

1. 配置 Turso/PostgreSQL/MySQL 数据库连接
2. 使用 WAE Database 模块提供的 ORM 功能
3. 配置数据库连接参数

## 相关模块

- `wae-database`: 数据库操作核心模块
